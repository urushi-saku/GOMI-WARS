# GOMI-WARS
アイデア1：地球規模のリアル陣取りゲーム「GOMI-WARS（ゴミ・ウォーズ）」
現実世界でゴミを拾い、その写真をアップロードすると、AIが「ゴミの種類と量」を判定。そのポイントに応じて、Googleマップ上のそのエリアが自チームの色に染まる、現実連動型の巨大陣取りWebアプリ。

企画アイディア: 「地球の清掃活動」という究極のソーシャルグッドを、「スプラトゥーンのような陣取りゲーム」というエンタメに変換する。

スケールの大きさ: 対象エリアは「地球全体」。世界中のユーザーが参加できるプラットフォームとして風呂敷を広げられます。

面白さ: 「あの駅前、敵チームに占領されてるからゴミ拾いに行ってくるわ」という、現実世界の行動を変容させる面白さ。

技術実装: * フロントエンド：Google Maps API（地図描画と色塗り）。

バックエンド：Firebase（位置情報とチームの陣地データをリアルタイム同期）。

AI：Gemini API等の画像認識（アップロードされた写真が本当にゴミか、何ポイント相当かを自動判定）。

影響範囲・完成度: デモ時に審査員にその場で「空き缶」の写真を撮ってもらい、会場周辺の地図の色がリアルタイムで変わる様子を見せれば、圧倒的なスケール感と完成度をアピールできます。
